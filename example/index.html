bugtoprompt standalone demo

This page has no React and no Tailwind. The floating bug-capture overlay in the bottom-right corner is injected entirely by the single <script> tag below — styles included.

Click Snap to open the panel, hit Record, click around, then Stop and copy/download the AI-ready prompt.

Config via data-* attributes on the script tag:

  • data-base — backend base URL (e.g. http://localhost:3000)
  • data-modes — comma-separated: clipboard,download,issue
  • data-project-id — project to file issues against
  • data-screenshot-mode — onMark | perPage | off
  • data-default-mode — primary action button
