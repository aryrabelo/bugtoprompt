# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1, as its code of
conduct. The full text is published at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: we want a friendly, respectful, and harassment-free community for
everyone, regardless of background or identity. Be kind, assume good intent,
and give constructive feedback.

## Scope

This code applies in all project spaces — the repository, issues, pull
requests, discussions — and whenever someone represents the project in public.

## Reporting

If you experience or witness unacceptable behavior, report it **privately** to
the project maintainer through the repository's private reporting channel
(GitHub **Security > Advisories** on
[github.com/aryrabelo/bugtoprompt](https://github.com/aryrabelo/bugtoprompt)).
Reports are handled confidentially and reviewed as promptly as possible.

Maintainers are responsible for clarifying standards and may take fair,
appropriate action — from a warning to a ban — in response to behavior they
deem inappropriate, following the enforcement guidelines in the linked
Contributor Covenant.
